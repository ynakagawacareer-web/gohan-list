# 近所のごはん帖 — セットアップ手順

ログイン不要・無料で家族全員が使えるアプリです。  
準備するのは **パソコン** と **Googleアカウント** と **GitHubアカウント（無料）** だけ。  
所要時間：約15分

---

## ファイル構成

```
index.html   ← アプリ本体（これをブラウザで開く）
gas_code.js  ← Googleスプレッドシート用のサーバーコード
README.md    ← この手順書
```

---

## STEP 1｜Googleスプレッドシートを作る

1. https://sheets.google.com を開く
2. 「空白のスプレッドシート」を作成（名前は何でもOK）
3. このタブはこのまま開いておく（次のステップで使う）

---

## STEP 2｜Google Apps Script を設定する

1. スプレッドシートのメニューから  
   **「拡張機能」→「Apps Script」** をクリック

2. 開いたエディタの中身（function myFunction...）を**全部消す**

3. `gas_code.js` をテキストエディタで開き、中身を**全部コピー**してエディタに貼り付ける

4. **Ctrl+S（Mac: ⌘+S）** で保存

5. 上部の **「デプロイ」→「新しいデプロイ」** をクリック

6. 左上の歯車アイコン ⚙ をクリックして **「ウェブアプリ」** を選択

7. 以下のように設定する：
   - 説明：（空欄でOK）
   - 次のユーザーとして実行：**「自分」**
   - アクセスできるユーザー：**「全員」**

8. **「デプロイ」** ボタンをクリック

9. 「アクセスを承認」と出たら：
   - Googleアカウントを選択
   - 「詳細」→「〇〇（安全ではないページ）に移動」をクリック
   - 「許可」をクリック

10. 「ウェブアプリのURL」欄に表示された  
    `https://script.google.com/macros/s/xxxxxxx/exec`  
    という形のURLを**コピーする**（次のステップで使う）

---

## STEP 3｜index.html にURLを貼る

1. `index.html` をテキストエディタで開く  
   （メモ帳 / VS Code / テキストエディット など何でもOK）

2. ファイルの上の方にある以下の行を探す：

   ```
   const GAS_URL = "YOUR_GAS_URL_HERE";
   ```

3. `YOUR_GAS_URL_HERE` をSTEP2でコピーしたURLに書き換える：

   ```
   const GAS_URL = "https://script.google.com/macros/s/xxxxxxx/exec";
   ```

4. 保存する（Ctrl+S）

5. **動作確認：** `index.html` をダブルクリックしてブラウザで開く  
   お店を1件追加してみて、Googleスプレッドシートに行が追加されれば成功！

---

## STEP 4｜GitHub Pages で公開する

### GitHubアカウントを作る（初回のみ）
1. https://github.com にアクセスして「Sign up」
2. メールアドレスとパスワードを設定するだけ（無料）

### リポジトリを作ってアップロード
1. ログイン後、右上の **「＋」→「New repository」**

2. 以下を入力：
   - Repository name：`gohan-cho`（何でもOK）
   - Public を選択（これを選ばないと公開されない）
   - 「Create repository」をクリック

3. 「uploading an existing file」リンクをクリック

4. `index.html` を画面にドラッグ＆ドロップ

5. 下の「Commit changes」をクリック

### Pages を有効にする
1. リポジトリページ上部の **「Settings」** タブをクリック
2. 左メニューの **「Pages」** をクリック
3. Branch の下のドロップダウンを **「main」** に変更して **「Save」**
4. 1〜2分待つと「Your site is published at ...」と表示される

---

## STEP 5｜家族に共有する

表示されたURL（例：`https://あなたのID.github.io/gohan-cho/`）を  
**LINEやメッセージで家族に送るだけ。**

- スマホのブラウザでそのまま使えます
- Googleアカウントも GitHubアカウントも家族には不要
- 誰かがお店を追加・更新すると、全員のページに反映されます
  （反映には数秒かかる場合があります）

---

## うまくいかないときは

**「アクセスできません」と出る**  
→ STEP2の手順9（承認）をもう一度確認。「全員」になっているか確認。

**お店が追加できない**  
→ GAS_URLが正しくコピーできているか確認（末尾に `/exec` がついているか）。

**スプレッドシートに反映されない**  
→ GASを再デプロイ（「デプロイを管理」→既存のデプロイを選んで更新）してみる。
